Paso a paso. (Si encuetras problemas favor de reportarlos como issues. No se probo en WIN)
Queres probarlo y no sabes como? Aca tenes los pasos:
	-Si estas en windows tenes que instalar Python, es gratuito y directo
		https://www.python.org/downloads/
	-Descargar estas dependencias escribiendo en la terminal:
		-Abrir terminal:
			Presionar tecla 'WIN'
			Escribir 'cmd' en el espacio y presionar 'ENTER'
		-En la terminal escribir:
			pip install numpy
			pip install tabulate
				-Si te causa error probar con: 'pip3' en vez de 'pip'
	-Una vez listas las dependencias podemos abrir la terminal en el directorio
		- Boton derecho en la carpeta (Abrir terminal)
		- tambien podemos copiar la direccion de la carpeta (direccion_copiada) y la pegamos en la terminal de esta forma:
			cd direccion_copiada

			-Puedes pegar en la terminal con boton derecho/pegar o CTRL+SHIFT+V

	-En la direccion del programa escribimos lo siguiente en la terminal:
		python3 pk.py



Ejecutar el pk.py escribiendo en la terminal: 'python3 pk.py'
